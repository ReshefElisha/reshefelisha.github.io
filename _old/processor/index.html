16-bit MIPS processor

Read More

My final project for Computer Organization (ECE 142) with Dr. Irvin Jones was to design and implement a 16-bit processor based on the MIPS architecture. The choice of implementation language was up to me, and I chose Python 2.7, as I am very familiar with it and some tricks that it has allowed me to write the entire thing in only 98 lines of code.

My implementation includes 10 functions:

  • ADDI: add an immediate value to a register's stored value. RA←RA+imm
  • BEQ: Branch to the specified immediate value if the given register stores the value 0.
  • NOP: The null operator, skips a cycle to let longer operations (memory operations) to happen safely.
  • LA: Load an address into the MAR (memory address register). This memory register is then used to load or store a value in memory. MAR←RA
  • LW: Load a word from memory. It loads the word stored in the address that the MAR stores and puts it into the given register. RA←MEM[MAR]
  • SW: Store a word in memory. It stores the word from the given register into the memory address set by the MAR. MEM[MAR]←RA
  • ADD: Adds two registers' values together and stores the new value into the first register. RA←RA+RB
  • SUB: Subtracts the value stored in the second register from the value stored in the first register, and stores the new value in the first register. RA←RA-RB
  • NOR: Performs the NOR binary operation on the values stored in the registers, and stores the new value in the first register. RA←RA↓RB
  • J: Jumps the Program Counter to an immediate value. PC←imm
